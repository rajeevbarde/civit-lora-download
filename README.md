## 🚀 How to begin after installation?

1. Share your LoRA folders path on harddrive
2. Press Scan Button
3. Register lora
4. Validate files
5. Fix duplicates
6. Fix Orphans
7. Download new LoRA
8. Fetch Metadata for trigger words
